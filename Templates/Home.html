1. Grid layout
Viele Webseiten basieren auf einer "grid" Ansicht, was bedeutet, dass die Seite in Spalten unterteilt ist: Eine reaktionsschnelle "grid" Ansicht hat häufig 12 Spalten und eine Gesamtbreite von 100%. Sie wird verkleinert und vergrößert, wenn Sie die Größe des Browserfensters ändern. Grid layout docs

2. Flexbox
Das Flexible Box Layout-Modul erleichtert das flexible Layout-Strukturen ohne Verwendung von Float oder Positionierung. Flex box docs

3. Bootstrap grid system
Das Bootstrap "grid system" verwendet eine Reihe von Containern, Zeilen und Spalten zum Layouten und Ausrichten von Inhalten. Es ist mit Flexbox gebaut und ist "fully responsive". Bootstrap grid docs

4. Pixels
Pixels sind "absolute length units"
Das "device pixel ratio"(oder CSS Pixel Ratio) ist wie das CSS das resolution vom Gerät interpretiert. Ein device pixel ist das physical pixel. Ein CSS pixel ist ein logisches pixel.
Absolute length units sind nicht empfohlen, weil die Bildschirmgrößen sehr unterschiedlich sind. Sie können jedoch verwendet werden, wenn das Ausgabemedium bekannt ist, z. B. für das Drucklayout.

5. %, EM, REM, VH, VW
"Relative length units" geben eine Länge relativ zu einer anderen Längeneigenschaft an.
% - Relativ zum übergeordneten Element
em - Relativ zur Schriftgröße des Elements (2em bedeutet zweimal die Größe der aktuellen Schrift)
rem - Relativ zur Schriftgröße des Root-Elements
vh - Relativ zu 1% der Höhe des "Viewport"
vw - Relativ zu 1% der Breite des "Viewport"*
vmin - 1vmax = 1vw or 1vh, abhängig von was kleiner ist
vmax - 1vmax = 1vw or 1vh, abhängig von was größer ist
