1. Grid layout
Many web pages are based on a grid-view, which means that the page is divided into columns: A responsive grid-view often has 12 columns, and has a total width of 100%, and will shrink and expand as you resize the browser window.
2. Flexbox
The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.
3. Bootstrap grid system
Bootstrap grid system uses a series of containers, rows, and columns to layout and align content. It is built with flexbox and is fully responsive.
